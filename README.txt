PanelProLanka Website
=====================

Files:
- index.html
- about.html
- services.html
- projects.html
- contact.html
- css/styles.css
- js/main.js
- assets/logo.jpeg

How to run:
1. Open index.html in a browser.
2. Or host the folder on any web server.

Customize:
- Replace phone/email in contact.html and footer sections.
- Add real photos into assets and update project cards.
- Change colors in css/styles.css root variables.
